# Make symbolic link files

\* Make sure the library path -> $HOME/mydot (symbolic link recommended)
```sh
$ ln -s [YOUR LIBRARY PATH] ~
$ cd symbolic-linker
$ . .symbolic-link-generate
```

# Install brew package list
